# MovieChallengeApp 🎬

MovieChallengeApp, The Movie Database (TMDB) API kullanılarak geliştirilmiş bir Android film keşif uygulamasıdır.  
Bu proje, bir mobil geliştirme challenge’ı kapsamında hazırlanmıştır ve modern Android geliştirme yaklaşımlarını göstermeyi amaçlamaktadır.

Uygulama Jetpack Compose ile geliştirilmiş olup MVVM mimarisi kullanılmıştır.

---

## 📱 Uygulama Özellikleri

- Ana sayfada farklı film kategorileri listelenmektedir:
  - Now Playing
  - Popular
  - Top Rated
  - Upcoming
- Film afişlerine tıklandığında film detay sayfası açılır
- Film detay sayfasında:
  - Film adı
  - Açıklama
  - Çıkış tarihi
  - Puan bilgileri gösterilir
- Film adına göre arama yapılabilir
- “Tümünü Göster” ekranı ile kategori bazlı tüm filmler listelenebilir
- İnternet bağlantısı olmadığında kullanıcı bilgilendirilir
- Alt navigasyon menüsü:
  - Ana Sayfa
  - Arama
- Koyu Mod / Açık Mod desteği bulunmaktadır

---

## 🛠 Kullanılan Teknolojiler

- Kotlin
- Jetpack Compose
- Retrofit
- Coil
- Material / Material3
- MVVM Architecture
- TMDB API
