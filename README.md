# MovieChallengeApp 🎬

MovieChallengeApp, The Movie Database (TMDB) API kullanılarak geliştirilmiş bir Android film keşif uygulamasıdır.  
Bu proje, bir mobil geliştirme challenge’ı kapsamında hazırlanmış olup modern Android geliştirme yaklaşımlarını göstermeyi amaçlamaktadır.

Uygulama Jetpack Compose ile geliştirilmiş ve MVVM mimarisi kullanılmıştır.

---

## 📱 Uygulama Özellikleri

- Ana sayfada farklı film kategorileri listelenmektedir:
  - Now Playing (Şu Anda Oynatılıyor)
  - Popular (Popüler)
  - Top Rated (En Yüksek Puanlı)
  - Upcoming (Yakında)

- Filmlere tıklandığında film detay sayfası açılır
- Film detay sayfasında:
  - Film adı
  - Açıklama
  - Puan bilgisi

- Film adına göre arama yapılabilir
- “Tümünü Göster” ekranı ile kategorilere ait tüm filmler listelenebilir
- İnternet bağlantısı olmadığında kullanıcı bilgilendirilir
- Alt menü:
  - Ana Sayfa
  - Arama
- Koyu Mod / Açık Mod desteği mevcuttur

---

## 🛠 Kullanılan Teknolojiler

- Kotlin
- Jetpack Compose
- Retrofit
- Coil
- Material / Material3
- MVVM Architecture
- TMDB API
