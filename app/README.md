# Movie Challenge App 🎬

Bu proje Android (Jetpack Compose) kullanılarak geliştirilmiş bir film keşif uygulamasıdır.

## 🚀 Özellikler
- Güncel filmleri listeleme (Now Playing, Popular, Top Rated, Upcoming)
- Film detay ekranı
- Film arama (Search)
- İnternet bağlantısı yokken uyarı gösterimi
- Dark / Light Mode desteği

## 🛠 Kullanılan Teknolojiler
- Kotlin
- Jetpack Compose
- Retrofit
- Coil
- Material Design 3

## 🌐 API
- The Movie Database (TMDB) API
